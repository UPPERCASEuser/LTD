LTD Docs Light Theme Disabler Docs

Disabler


--disabler-text

Set the text of the disabler.
Type: String.
Original Value: "Please disable light theme.\aYour theme doesn't support light theme"

--disabler-bg

Set the background color of the disabler.
Type: Color
Original Value: Black

--disabler-color

Set the color text of the disabler.
Type: Color
Original Value: White



Theme changer alert


--opt-text

Set the text of no supporting text alert.
Type: String.
Original Value: " Disabled because your theme doesn't support light theme"

--opt-text-color

Set the text color of no supporting text alert.
Type: Color
Original Value: Red

--opt-opacity

Set the opacity of the theme light switcher.
Type: opacity
Original Value: 0.6
