LTD Docs Light Theme Disabler Docs

--disabler-text

Set the text of the disabler.
Type: String.
Original Value: "Please disable light theme.\aYour theme doesn't support light theme"

--disabler-bg

Set the background color of the disabler.
Type: Color
Original Value: Black

--disabler-color

Set the color text of the disabler.
Type: Color
Original Value: White


--warning-color

Set the color of red dashed borders and other stuff.
Type: Color
Original Value: Red

--alert-color

Set the color of the text that says that a plugin doesn't support the theme
Type: Color
Original Value: Yellow

--disabler-text

Set the text that says a plugin doesn't support the theme
Type: String.
Original Value: "Not supported for your actual theme. Please disable it"
